# CafeX frontend

The frontend uses a familiar Vite-style `src` structure for application code,
with a minimal `app` adapter retained for CafeX hosting.

```text
frontend/
├── public/
├── src/
│   ├── assets/
│   ├── component/
│   ├── data/
│   ├── pages/
│   ├── service/
│   ├── App.css
│   ├── App.jsx
│   ├── index.css
│   └── main.jsx
└── app/
    ├── globals.css
    ├── layout.tsx
    └── page.tsx
```

See the root `README.md` for MySQL Workbench, backend, admin-account, and startup instructions.
